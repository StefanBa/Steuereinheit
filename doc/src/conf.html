File ..\src\conf.lua

Author:

Release: Datum: 17.08.2012

Functions

default ()
Füllt die globale Tabelle "const" mit default-Werten.
execute ()
Speichert die globale Tabelle "const" mittels der Funktion "serialize" auf die SD-Karte
get (id, format)
Gibt ein Eintrag der "const" Tabelle zurück. Ist der Eintrag eine Tabelle, so werden default mehrere Rückgabewerte erzeugt. Es kann auch die Tabelle selbst verlangt werden.

Parameters:

  • id: Name des Eintrages
  • format: Tabelle als Rückgabewert wenn "*t"

Return value:

    Gespeicherte Werte vom entsprechenden Eintrag.
init ()
Initialisiert die persistenten Werte entweder von der SD-Karte. Sind diese nicht verfügbar, werden die default-Werte verwendet.
serialize (o, arg)
Serialisiert eine Zahl, ein String oder eine Tabelle und Speichert das Resultat in das momentan geöffnete file --> boardconf.lua. Kann auch mit verschachtelten Tabellen umgehen, wird dann rekursiv verwendet.

Parameters:

  • o: zu serialisierendes Objekt
  • arg: optional, zusätzliches Argument um Formatierung zu ermöglichen
set (id, value)
Erzeugt oder verändert ein Eintrag in der "const" Tabelle und aktualisiert File auf der SD-Karte.

Parameters:

  • id: Name des Wertes
  • value: zu speichernder Wert
