Functions

checksum (s)
Erzeugt die Checksumme eines Strings und gibt diese zurück

Parameters:

  • s: String zur erzeugung der Checksumme

Return value:

    Checksumme
readFromFile (file)
Liest Daten der grösse BLOCKSIZE aus file und sendet diese auf der UART-Schnittstelle. Vorgängig wird die Checksumme berechnet und als letztes Byte angehängt.

Parameters:

  • file: file, von dem gelesen wird
recv (filename, filesize)
Wird aufgerufen um ein file zu empfangen. Erstellt oder Öffnet (-> Überschreibt) das file, berechnet Anzahl Pakete und führt entsprechend "writeToFile" aus.

Parameters:

  • filename: Name des files
  • filesize: Grösse des files

Return value:

    boolean true, falls Übertragung erfolgreich
send (filename, filesize)
Wird aufgerufen um ein file zu senden. Versucht das file zu öffnen, berechnet die Anzhal Pakete und ruft entsprechend "readFromFile" aus.

Parameters:

  • filename: Name des files
  • filesize: Grösse des files

Return value:

    boolean true, falls Übertragung erfolgreich
writeToFile (file, difflen)
Wartet auf UART-Daten der Grösse BLOCKSIZE und schreibt diese in das file. Überprüft zuerst ob die Checksumme der Datenbytes (alle ausser das letzte) dem Checksummenbyte (letztes Byte) entspricht.

Parameters:

  • file: file, in das geschrieben wird
  • difflen: optional, für letzter Block, falls dieser mit Nullen aufgefüllt wurde
