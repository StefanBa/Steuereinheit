Functions

merge ()
Virtualisierung der IOs. Falls das Feld "custom" eines IO's ein Wert hat, so wird der custom-Wert ins Feld "merge" geschrieben. Ansonsten erhält das Feld "merge" der reale Wert.
reset (key)
Setzt die Werte eines bestimmten Feldes der IO's zurück.

Parameters:

  • key: Feld des IOs ("real", "custom" oder "merge")
run ()
Funktion der Koroutine. Führt jeweils merge und update aus, um die IOs ständig ein-, beziehungsweise auszulesen.
update ()
Aktualisiert die Werte der Ein- und Ausgänge je nach Typ.
